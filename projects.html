AI-Driven Plant Analysis

Identifying and segmenting root structures using Computer Vision

Client: The Netherlands Eco-phenotyping Centre (NPEC)

Learn More

The Netherlands Plant Eco-phenotyping Centre (NPEC) is dedicated to advancing plant research for sustainable agriculture. One of the key challenges in this field is automating root segmentation from images to achieve accurate root detection and analysis. Additionally, robotics plays a crucial role in precise plant inoculation. This project addresses these challenges by developing a robust plant segmentation approach, accurate root tip detection, and precise inoculation using a prototype of the OT-2 robot.

To tackle the challenges of plant phenotyping, computer vision methods were used to separate Petri dishes from background interference and perform semantic segmentation on plant elements like seeds, shoots, and roots. After refining and preprocessing the dataset, I created a machine learning model that could accurately predict masks for various plant structures. This enabled instance segmentation, which identified individual plants within the images, allowing for precise measurements such as root length and the exact location of root tips—a crucial factor in evaluating plant growth and characteristics.

In parallel with the computer vision work, the robotics aspect of the project focused on automating the delivery of inoculants to the identified plant root tips. I simulated and developed a precision liquid handling robot that was programmed using a PID controller for accurate liquid dispensing. The robot was integrated with the computer vision pipeline, enabling it to locate and deliveer the liquid to targeted areas on the Petri dish. This demonstarted how vision-based analysis could be seamlessly combined with robotic automation, allowing for precise interventions in plant phenotyping experiments.

Beside the PID controller, I also developed a Reinforcement Learning model. By designing appropriate reward functions and conducting hyperparameter tuning, I ensured that the robot could autonomously navigate to the correct root tips for liquid delivery. By comparing the PID controller and the Reinforcement Learning-based control system, I conducted a comparative analysis between these 2 systems. The PID controller demonstarted superior accuracy when combined with the Computer Vision pipeline, achieving precision below 1mm. However, with Reinforcement Learning, the robot performed the task way faster, but with a little bit worse accuracy.

Project Highlights:

  • Implemented a U-Net model achieving an F1 score of 0.85 for root mask prediction.
  • Applied Dijkstra's shortest path method to accurately measure primary root length.
  • Developed and compared a Reinforcement Learning model and a PID controller for precise inoculation using a simulated OT-2 robot. While RL was faster, the PID controller achieved higher accuracy in targeting root tips.

Key Takeaways:

  • Strengthened expertise in Computer Vision for agriculture applications.
  • Gained fundamental knowledge of robotics, automation, and precision control.
  • Designed and developed an end-to-end AI-driven solution, from concept to working prototype.
